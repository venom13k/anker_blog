Добавление пользователя Windows в домен Samba с помощью LDAP

** Добавление пользователя smbldap-useradd -a -c "Samba User" -d /dev/null -g 513 -u 8081 -s /bin/false -C "" -D "" -E "" -F "" -N "USERNAME" USERNAME

Назначение пароля в ldap smbldap-passwd USERNAME При неудачи назначить отдельно одинаковые пароли для Samba и UNIX пользователя: * Назначение пароля Samba smbpasswd USERNAME

*** Назначение пароля Posix passwd USERNAME

** Добавление пользователя в DNS-сервер BIND Добавить имя хоста и IP-адрес в файл на сервере /etc/bind/db.intranet и /etc/bind/db.intranet.revers Также поменять порядковый номер Serial на текущую дату и порядковый номер

Установка скрипта для пользователя Скрипт для пользователя изменить с шаблона в каталоге /usr/samba/netlogon/USERNAME.bat Применение групповых политик Изменить с шаблона /usr/samba/netlogon/policy/USERNAME.pol и применить на компьютере пользователя, изменив расширение на USERNAME.reg

** Применение настроек на компьютере пользователя Применить файлы реестра на компьютере пользователя, лежащие на сервере:

smb://server/stuff/ossystem/Update/WinXP/autodisconnect.reg smb://server/stuff/ossystem/Update/WinXP/cp1251.reg smb://server/stuff/ossystem/Update/WinXP/dos_kbd/reg smb://server/stuff/ossystem/Update/WinXP/noautorunnew.reg smb://server/stuff/ossystem/Update/WinXP/terminalserver.reg smb://server/stuff/ossystem/Update/WinXP/WinXP_SignOrReal.reg
