Завод АНКЕР получил подтверждение действия Сертификата соответствия системы менеджмента качества требованиям ГОСТ Р ИСО 9001- 2008( ИСО 9001:2008 )

6 мая 2011 года Орган по сертификации систем качества АНО " Центр менеджмента качества и сертификации " ( № РОСС RU.0001.13ИС56 ) подтвердил действие сертификата, выданного Заводу АНКЕР.

Регистрационный номер сертификата в Регистре Систем Качества системы сертификации Госстандарта России
