Поздравляем с Днем Металлурга!

Челябинский " Завод АНКЕР " был создан как специализированное предприятие- поставщик металлоконструкций и оборудования для коксохимического производства металлургической отрасли. Происхождение яркого имени- Завод АНКЕР- связано с названием одного из основных видов продукции Завода для металлургической отрасли- анкеража( каркаса коксовых батарей- печей для изготовления кокса ) . Поэтому День металлурга- праздник, который отмечают клиенты, стал и нашим.
